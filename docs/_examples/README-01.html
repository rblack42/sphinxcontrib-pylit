sphinxcontrib-pylit¶

Sphinx extensions for Literate Programming.

This extension supports a version of Dr. Donald Knuth’s Literate Programming within reStructuredText documents. Code detailed within blocks supported by this extension can be extracted for processing, and saved in a user-configurable location. The extension also supports running commands against the extracted code. This extension was developed to support teaching Computer Science and demonstrating the software development workflow.
