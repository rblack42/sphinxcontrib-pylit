The PyLiT Sphinx_extension¶

The sphinxcontrib-pylit package is a Sphinx extension providing a domain with several new directives that support Literate Programming. It also provides a command-line interface for manipulating the code documented in reStructuredText files. This section contains auto-generated documentation produced by the Sphinx autodoc tools.
