Welcome

Dr. Donald Knuth:

“I believe that the time is ripe for significantly better documentation of programs, and that we can best achieve this by considering programs to be works of literature. Hence, my title: Literate Programming.“

PyLiT is an extension to Sphinx that allows a variation of Dr. Knuth's Literate Programming concept to be included in reStructuredText documentation. The tool provides support for extracting code from the documentation and placing thta code in a git repository, complete with a pseudo history making it usable with normal git tools. PyLiT was written by Roie R. Black and licensed under the BSD license.
