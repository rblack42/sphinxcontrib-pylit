pylit-quickstart
================

Synopsis
--------

**pylit**

Description
-----------

:program:`pylit` is a command-line tool that provides access to project code written using the |LP| extensions provided by **sphinxcontrib-pylit**.

Options
=======

..  program:: pylit

..  option::    -a, --author

    Display the project author and email.

..  option::    -v, --version

    Display the current version of this program.

See also
--------
