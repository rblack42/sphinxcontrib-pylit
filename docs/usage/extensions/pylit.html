sphinxcontrib.pylit – Support for Literate Programming

Read time: 1.0 minutes (97 words)

This extension creates a Sphinx domain with these new directives:

.. pylit.code::¶

This directive marks the starting block of a list of code blocks. This list can be extended using the pylit.add directive (see below).

.. pylit:add::¶

This directive references a previously defined code Literate Programming block. The code in this block is appended to the list of blocks currently contained in the named code block.

.. pylit.file::¶

Use this directive to identify a new file containing the named code block list. the code written to this file will include the current list of blocks contained in the nammed Literate Programming block.
